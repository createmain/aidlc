# 테이블오더 서비스 - 사용자 페르소나

---

## Persona 1: 고객 (Customer)

- **이름**: 김고객
- **역할**: 매장 테이블 이용 고객
- **주요 목표**: 대기 없이 빠르고 편리하게 메뉴를 탐색하고 주문하기
- **사용 환경**: 테이블에 설치된 태블릿 (브라우저 기반 웹 UI)
- **관련 스토리**: US-C01 ~ US-C06

---

## Persona 2: 관리자 (Admin)

- **이름**: 박관리자
- **역할**: 매장 운영자 / 관리자
- **주요 목표**: 실시간으로 주문을 모니터링하고, 테이블과 메뉴를 효율적으로 관리하기
- **사용 환경**: 관리자 PC 또는 태블릿 브라우저
- **관련 스토리**: US-A00 ~ US-A09
